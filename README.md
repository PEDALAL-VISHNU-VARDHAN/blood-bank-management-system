# Blood Bank Management System
A Smart Blood Bank Management System built using Flask, SQLite, HTML, CSS, Bootstrap, and Python.
This project helps manage blood donors, donation eligibility, health verification, feedback management, and admin operations through a modern web-based dashboard.
---
#  Features
## User Features
- User Registration & Login
- Blood Donor Dashboard
- Donation Availability Toggle
- Donation Eligibility Checking
- Donation Interval Tracking
- Health Status Monitoring
- Coins Reward System
- Upload Blood Reports
- Submit Feedback with Images
- View Donation Status
---
##  Admin Features
- Admin Login System
- Donor Verification
- Health Verification Dashboard
- Blood Report Review
- Feedback Management
- Coin Management
- Donation Interval Management
- Search Donors
- Dashboard Controls
